## 📌 Sobre o Projeto
O **Apex Cinema** é uma aplicação interativa desenvolvida para explorar e analisar dados do setor cinematográfico. O objetivo deste projeto é transformar um grande volume de dados brutos sobre filmes em visualizações dinâmicas e fáceis de entender, facilitando a descoberta de tendências da indústria.

Este projeto foi construído para colocar em prática os estudos de manipulação de dados e criação de dashboards.

## 🚀 Funcionalidades
O painel oferece uma interface amigável com os seguintes filtros interativos:
* **Classificação:** Filtre filmes por faixa etária ou notas.
* **Gênero:** Descubra tendências em categorias como Ação, Comédia, Drama, etc.
* **Ano de Lançamento:** Analise a evolução do cinema ao longo das décadas.
* **País de Origem:** Explore as produções cinematográficas ao redor do mundo.

## 🛠️ Tecnologias Utilizadas
A construção deste projeto envolveu as seguintes tecnologias e bibliotecas:
* **Python:** Linguagem principal utilizada na lógica do projeto.
* **Pandas:** Essencial para a limpeza, tratamento e estruturação da base de dados original.
* **Streamlit:** Framework utilizado para a criação da interface web interativa com Python.
* **Power BI:** Utilizado para prototipação e construção de visualizações avançadas de dados.
